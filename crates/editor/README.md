# editor

Editor UI panels — the workspace chrome rendered around the engine. These were `ui-crates/` before the restructure.

| Crate | Purpose |
|---|---|
| `ui_about` | About / credits dialog |
| `ui_common` | Shared UI widgets, styling, i18n |
| `ui_core` | Core editor shell — plugin loading, layout, menus |
| `ui_documentation` | In-editor documentation viewer |
| `ui_entry` | Application entry point and window setup |
| `ui_fab_search` | Floating action button with omnibox search |
| `ui_file_manager` | File tree and project browser |
| `ui_flamegraph` | CPU/GPU profiling flamegraph viewer |
| `ui_friends` | Friends list and social panel |
| `ui_git_manager` | Git integration panel |
| `ui_level_editor` | Level/scene editor panel |
| `ui_loading_screen` | Splash screen and loading progress |
| `ui_log_viewer` | Log output viewer |
| `ui_multiplayer` | Multiplayer session UI |
| `ui_multiuser_status` | Multi-user presence indicators |
| `ui_plugin_manager` | Plugin browser and installer |
| `ui_problems` | Errors / warnings / lint panel |
| `ui_settings` | Settings editor |
| `ui_type_debugger` | Runtime type inspector |
| `ui_types_common` | Shared type definitions for UI crates |
