# ui

Vendored UI framework repos, added as git submodules and referenced as path deps from the root workspace.

| Submodule | Path dep for |
|---|---|
| `wgpui/` | `gpui-ce` — community fork of Zed's GPU-accelerated UI framework |
| `wgpui-component/` | `ui` + `ui-macros` — rich UI primitives built on GPUI |

Changes to these submodules are committed directly in the Pulsar-Native tree.
